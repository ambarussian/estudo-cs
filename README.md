# Estudo sobre C#

## Introdução
Este estudo tem como objetivo explorar a linguagem C# por meio da resolução de exercícios práticos, organizados por temas e distribuídos em pastas específicas.

---

## Objetivos
- Compreender o funcionamento da linguagem C#;
- Aprender e aplicar conceitos fundamentais, como matrizes, classes, encapsulamento, herança, herança por implementação, associações e polimorfismo, utilizando a sintaxe da linguagem.

---

## Arquitetura
- `atv1_conhecendo_cs`
- `atv2_matrizes`
- `atv3_classes`
- `atv4_encapsulamento`
- `atv6_associacoes`
- `atv7_heranca_por_implementacao`
- `atv8_polimorfismo`

Cada pasta contém exercícios relacionados ao respectivo tema. No entanto, como este estudo foi realizado há algum tempo, o diretório denominado `atv5_heranca` foi corrompido e, infelizmente, não pôde ser recuperado.
